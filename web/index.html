Instructions for the chat demo:

  • Click the chat image to open the chat
  • Start by saying Hi or Hello
  • Search sites by id or choose from the list
  • Id's 1-4 will work (rows in database), other id's return not found/try again
  • Id retry count is 2 if a non numeric id is entered
  • When found (or selected from the list) site details are shown including it's status
  • Depending on site status site can be activaed or deactivated
  • Alternaively can end chat just by choosing finish
